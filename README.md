# Hi, I'm Alex Spigler 👋

I'm a junior at George Washington University pursuing a **BS in Statistics** and a **BA in Computer Science** with a minor in Mathematics. My work sits at the intersection of statistical modeling, computational methods, and data-driven decision making.

## What I'm Working On

**[Chaotic Attractors Toolkit](https://github.com/alexspigler/chaotic-attractors)** — A Python package for computational exploration of chaotic dynamical systems. Searches parameter space across 9 attractor families (5 classical, 4 custom) using stochastic sampling and multi-stage statistical filtering to identify chaotic regimes, then applies Gaussian KDE to reveal underlying fractal structure. Production-ready with 50+ unit tests and a CLI interface.

**[Time Series Structural Break Analysis](https://github.com/alexspigler/time-series-structural-break-analysis)** — An intervention analysis detecting competitive market impact using regression with ARIMA(1,0,1) errors. Includes a full diagnostic testing suite, 10-week forecasts with prediction intervals, and a reproducible R Markdown report.
